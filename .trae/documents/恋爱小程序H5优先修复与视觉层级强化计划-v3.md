> **执行原则**：H5 优先调试 → mp-weixin 适配 → agent-browser 真实浏览器验证 → 最终审查
> **范围**：全部 11 个主页面 + 公共组件库 + stores + 配置文件
> **匹配概率**：80%（用户明确指定）
> **决策深度**：底层重构清理，真实审查完全

## 一、Summary（摘要）

基于真实代码审查，本计划针对用户反馈的 8+4 类问题给出 4 阶段修复方案。所有问题已在 Phase 1 探索阶段验证，根因明确：

1. **P0 致命 Bug**：`discover/index.vue` 第 272 行多余 `</view>` 闭合标签，导致后续 `daily-question-card`、`error-banner`、`loading-state`、`card-area` 等元素被错误嵌套或隐藏 — 这是「按钮匹配异常、点击进入后内容消失」的真正根源。
2. **P0 匹配概率**：`stores/discover.ts` 第 647 行和第 707 行均为 `Math.random() < 0.3`（30%），不符合用户要求的 80%。
3. **P0 签到动画窗口过长**：`stores/checkin.ts` 第 309 行 `setTimeout(..., 3000)`（3 秒），导致「签到标签不显示」的视觉错觉（权益卡片延迟 3 秒才出现）。
4. **P1 按钮反馈不足**：`Button.vue` 仅有 `:active { transform: scale(0.96) }`，`CardSwiper.vue` 仅有 `scale(0.85)`，均无振动反馈、无 brightness 滤镜、无 press-feedback 工具类，mp-weixin 下 `:active` 伪类不可靠。
5. **P1 CardSwiper 图片渲染问题**：用原生 `<image>` 而非 `SafeImage`，无骨架屏、无失败降级；底部按钮用 emoji（✕ ⭐ ❤️），跨设备渲染不一致；`iconSrc` 常量为死代码；`e.preventDefault()` 在 mp-weixin 不工作。
6. **P1 首页 emoji 替代真实图片**：`home/index.vue` 第 31-41 行 11 个 `ICON_*` 硬编码常量是死代码；第 406/468/483/524 行用 emoji 替代真实图片；`posts[i].images` 和 `shopItems[i].image` 路径已存在但模板循环未消费。
7. **P1 profile 菜单功能缺失**：7 个主菜单中 4 个为「敬请期待」toast（我的收藏/我的匹配/访客记录/恋爱认证），3 个跳转路径不存在（`/subpackages/profile/posts/index`、`/subpackages/profile/lab/index`、`/subpackages/profile/settings/index`）；`conic-gradient` 和 `backdrop-filter` 在 mp-weixin 不兼容。
8. **P1 视觉层级与边缘**：缺少明确的 z-index 体系、边缘色变量、卡片分隔系统。

#### 1.4 创建振动反馈工具 utils/haptic.ts【P1】

**文件**：`src/utils/haptic.ts`（新建）

**作用**：封装 `uni.vibrateShort` 调用，H5 端静默失败，mp-weixin 端真实振动，统一调用入口。

```ts
/**
 * 振动反馈工具
 * - H5 端：静默失败（uni.vibrateShort 在 H5 不支持，但不会抛错）
 * - mp-weixin 端：触发短振动（15ms 类型）
 * - App 端：触发短振动
 *
 * 用法：
 *   import { lightHaptic, mediumHaptic, successHaptic } from '@/utils/haptic';
 *   lightHaptic();        // 轻量振动（按钮点击）
 *   mediumHaptic();       // 中等振动（卡片切换）
 *   successHaptic();      // 成功振动（签到成功、匹配成功）
 */

/** 轻量振动：按钮点击、Tab 切换 */
export function lightHaptic(): void {
  // #ifdef H5 || APP-PLUS || MP-WEIXIN
  try {
    uni.vibrateShort({ type: 'light' } as any);
  } catch {
    // 静默失败：H5 端不支持
  }
  // #endif
}

/** 中等振动：卡片切换、滑动操作 */
export function mediumHaptic(): void {
  // #ifdef H5 || APP-PLUS || MP-WEIXIN
  try {
    uni.vibrateShort({ type: 'medium' } as any);
  } catch {
    // 静默失败
  }
  // #endif
}

/** 重振动：飞出动画触发 */
export function heavyHaptic(): void {
  // #ifdef H5 || APP-PLUS || MP-WEIXIN
  try {
    uni.vibrateShort({ type: 'heavy' } as any);
  } catch {
    // 静默失败
  }
  // #endif
}

/** 成功振动：签到成功、匹配成功（连续两次轻振动） */
export function successHaptic(): void {
  lightHaptic();
  setTimeout(() => lightHaptic(), 100);
}
```

**验证**：H5 端调用不报错；mp-weixin 端在真机触发振动。

### 4.2 技术决策

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 振动 API | `uni.vibrateShort({ type })` | 跨端兼容，H5 静默失败 |
| 按压反馈 | `hover-class` + `press-feedback` 工具类 | mp-weixin `:active` 不可靠 |

| 图片路径 | `IMAGE_PATHS` 集中管理 | 避免硬编码 |
| CSS 降级 | `linear-gradient` + `box-shadow` 替代 `conic-gradient` / `backdrop-filter` | mp-weixin 兼容性 |
| 触摸事件 | `@touchmove.stop.prevent` 替代 `e.preventDefault()` | mp-weixin 兼容性 |
| 入场动画 | 内联 `pageVisible` + `page-fade-in` 类 | memory 硬约束（禁止外部 composable） |
| 环境检测 | `isDev` 从 `services/env.ts` 导入 | memory 硬约束（禁止 import.meta.env.DEV） |

   - **风险**：profile 菜单新增 ICONS_PROFILE 分类需要下载 8 个新图标。

   - **验证**：uni-app 官方文档支持，H5 端不抛错。
   - **风险**：旧版微信基础库可能不支持 `type` 参数（仅 `style` 兼容）。

3. **假设**：H5 端 `hover-class` 等效于 `:active`。
