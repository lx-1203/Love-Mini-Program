## 设计特点
- 参考青藤之恋核心功能架构（学历认证、兴趣图谱、资料完善度）
- 天蓝色系主色调：清新、理性、校园感强，降低恋爱压迫感
- 差异化元素：
  - 学历认证徽章（青藤风格）
  - 兴趣图谱可视化
  - 恋爱测试模块
- 保留社交类小程序核心 UX 逻辑
