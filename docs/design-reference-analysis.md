## 一、设计细节分析（按图分述）

### 1.1 浅色主题总览 — [14_53_26.png](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_53_26.png)

整体框架：左上方为方案封面区（标题"**本校园限定的校园恋爱交友小程序**" + 副标题"**真实校园 · 附近的人 · 自然相遇**" + 描述文案"基于校园场景，连结同校同学的日常与兴趣。在活动、课表与日常交互中自然遇见" + 角标 "DESIGN PROPOSAL V1.0"），右上方为版本说明"**本校限定 / 当前仅限 华南理工大学**"，下方水平排布 5 块手机样机 + 1 个最近活动弹层 + 1 个会员页示意。

- **登录页（左侧第 1 屏）**：状态栏 9:41，下方主视觉是一张泛绿的校园图书馆/草地实景大图压底，叠加大字标题"Hello! / 在华工，遇见同校的你"，再下一行小字"**华南理工大学 · 本校限定**"。CTA 区为两枚按钮：主按钮"微信登录"使用青绿色实心（带微信图标），次按钮"手机号登录"为白色描边，再下方一行小灰字"《已有校缘并同意《用户协议》和《隐私政策》》。背景图为 4:3 留白式，主体与按钮的纵向节奏为「图像(70%) → 标题块(15%) → 按钮块(15%)」。
- **校园墙页（左侧第 2 屏）**：顶部 Tab "**校园墙** / 最新 / 关注"，右上"发一条"圆形深色 CTA。卡片采用白底圆角 16、左右内边距 16、内容为头像 + 用户标签（学院·年级）+ 时间戳 + 正文 + 3 张 1:1 图片九宫格 + 位置标签胶囊 + 底部互动行（点赞/评论/收藏）。两屏卡片垂直排布，卡片之间留白 12，列表视觉密度轻。
- **首页（中间第 3 屏·主屏）**：9:41 状态栏 → 校园定位选择器"本校限定 · 华南理工大学 ▾" + 搜索/铃铛 → 横向卡片区"**校园圈活动**"（含 3 张状态徽章分别为"报名中 / 进行中 / 预告"的图片卡，下含时间、地点、人数堆叠头像）→ 课表"**课表空闲**"以周视图呈现，色块为浅薄荷/浅蓝/浅紫/浅绿四色课程块，底部"查看本周课表 >"→ 校园墙"**最新 / 关注**"嵌入单条 → "**逛逛推荐**"商品/票务卡片用 9.9/18/0 等红色价格字。
- **聊天页（中间第 4 屏）**：顶部返回 + 昵称"**抢滨**" + 绿色"在线"小点 + 右上"..."，中部消息流呈左右气泡（对方浅灰底、本人浅蓝底），气泡圆角 18、内边距 10/14，气泡间贴 11:21 / 11:27 时间戳。底部粘性输入条：左侧语音 → 居中输入框（"输入消息"占位灰字）→ 表情/加号。
- **最近活动弹层（最右第 5 屏）**：黑色半透明遮罩 + 居中卡片，卡片顶部一张夜景活动图，下方"**最近活动**"标题 + 关闭"×"，再下"校园十佳歌手决赛"标题 + 时间/地点/参与人数 + 红色"¥18 起" + 蓝色胶囊"去看看"，卡片圆角 24，外侧有 3 个小圆点指示器表示是轮播首条。
- **我的会员页（最右第 6 屏）**：9:41 状态栏 + 返回 + "我的"标题 + 搜索/铃铛 → 头像 + 昵称"**夏天的风**" + "华南理工大学 · 本科 · 大三" 灰色元信息 → 横幅"**高级会员**"为深色卡片 + 右侧"立即续费"白色描边按钮 + 截止日"2025.06.20 到期" → 8 格 2 行宫格图标（**谁看过我 / 解锁信息 / 优先排名 / 隐身浏览**等）→ "**为你推荐**"标题 + 3 张人物横卡（头像 + 昵称 + 学院年级 + "打个招呼"描边按钮）。

**整体自然流畅之处**：① 浅色版采用"图片大块 + 白卡 + 浅灰背景"的三层纵向节奏，每屏均保持同一份间距语言（16/12/8）；② 主色出现在唯一一个高对比位置（CTA、状态徽章、价格字），其余均走灰度，承担"主色点缀"的角色；③ 卡片圆角统一为 16，按钮主按钮 24 胶囊、次按钮 12 描边，形成"卡片 16 / 弹层 24 / 胶囊 24 / 输入控件 12"四级圆角语言。

### 1.2 深色主题总览 A — [14_57_30.png](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_57_30.png)

整体框架：左侧大尺寸校园夜景实景图（操场跑道 + 教学楼亮灯）压底，叠 6 行"特性 + 描述"列表，每行左侧为一个直径约 32 的圆形线性图标（描边 1.5px、白色），右侧标题"**本校限定生态**"加描述"仅限华工同学加入"。右侧并排 4 块手机样机展示同一系统在深色模式下的表现。

- **特性清单（左 1/3）**：6 条均使用「图标 + 标题 + 一句说明」三段式，分别为"本校限定生态 / 校园墙 · 圈子 / 课表空闲 / 聊天主场 / 逛逛生活 / 高级会员"。图标采用统一描边白色 1.5px，包括盾牌/对话气泡/时钟/语音气泡/购物袋/皇冠——以线性而非填充为主，深色背景下保持高识别度。
- **登录页（深色版，第 1 屏）**：与浅色版同为校园实景压底，但底图换成夜间亮灯版，主标题字号、字重保持一致；"微信登录"按钮仍为青绿色实心，"手机号登录"为半透明白底按钮（白字 + 半透明白边），整体在深色场景下能保持"高亮绿"是唯一亮色的克制。
- **校园墙（深色版，第 2 屏）**：底色为深色 (#0E1116 估计)，但内部"**图书馆 · 三楼自习区**"等位置标签使用淡蓝胶囊、文字用白/灰 80% 双色；图片区在深色底中仍保有自然色温，没有强行反相。
- **聊天页（深色版，第 3 屏）**：气泡采用"对方 = 深灰圆角矩形 / 本人 = 主品牌青绿色圆角矩形"，在线状态绿点保持 1.5 半径小点，消息列表底部吸附输入条 + 表情与加号。
- **会员页（深色版，第 4 屏）**：高级会员横幅由浅色版的"白底深色文字"翻转为"金棕色渐变底 + 白字"（金棕色 #C9A36A → #E8C98A 渐变估计），立即续费按钮变为"白底棕字"实心，整体在深色版中更加显眼；8 格宫格与人物推荐区保持与浅色版一致的版式，仅作颜色反转。

**整体自然流畅之处**：① 深色版没有简单地把所有卡片底色变深，而是**只在登录/会员横幅两处做反相**，校园墙、聊天等"内容为主"的页仍保留"图片为主、文字为辅"的视觉权重；② 图标始终走线性、1.5px 描边、白色 100%，没有出现填充式彩色图标——这是克制感的关键；③ 暖色（青绿 CTA + 金棕 VIP 渐变）在整套深色界面中只出现 2 处，符合"80/20 主色点缀原则"。

### 1.3 深色主题总览 B（含特性清单）— [14_57_39.png](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_57_39.png) / [ref_img.png](file:///d:/6/恋爱小程序/ref_img.png)

整体框架：上方标题"**本校园限定 / 校园恋爱交友小程序**" + 副标"校园探索感 · 界面夜色 · 原生与细腻"，其下沿用 [14_57_30](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_57_30.png) 同样的 6 条特性清单，但**图标与说明位置互换到左侧垂直一列**，更像是"目录大纲"；右侧 9 块手机样机以 3×3 网格排布，密度更高。

- **特性清单（左列）**：与 A 版图标完全相同（盾牌/对话气泡/时钟/语音气泡/购物袋/皇冠），区别在于排列方向由横排改为纵列堆叠，每行高度 56~64、上下间距 8，文字在图标右侧 8px 起排。
- **登录页（深色）**：校园夜景图压底 + "在华南理工大学 / 遇见同校的你" 标题，"**本校限定 · 真实校园生态**" 副标；按钮仅露出"**微信一键登录** / 手机号登录"两枚，颜色同 A 版。
- **首页（深色）**：与浅色首页结构同构——"**校园圈活动** / **课表空闲** / **校园墙** / **逛逛推荐**" 4 个模块顺序不变，仅卡片底色深色化、价格字仍为红色 #E5454D 系，状态徽章（报名中/进行中/预告）保留 4 个状态色：薄荷绿 / 琥珀黄 / 浅蓝紫。
- **最近活动弹层（深色）**：与浅色版同结构，但卡片底变为深灰 + 顶部 1px 白色 8% 分隔线，"去看看"按钮从浅色版的蓝色胶囊改为青绿色胶囊，呼应主品牌色。
- **逛逛页（深色）**：顶部搜索条 + 5 枚横向分类胶囊（活动票/餐饮/周边/文创/全部），"**精选推荐**" 横向卡片 2 张，商品图 + 标题 + 时间 + 价格 + 销量；底部 TabBar 第 4 个"逛逛"图标高亮为青绿。
- **会员页（深色）**：与 A 版会员页一致，金棕渐变横幅；"**高级会员**"标题下还有 4 张特权卡（**查看过谁 / 无限喜欢 / 专属标识 / 优先推荐**）2×2 网格，最底部一枚"**立即开通**"主操作按钮（金棕渐变实心 + 白字 + 100% 圆角胶囊）。

**整体自然流畅之处**：① 通过把"6 条特性"做成纵向目录，把整套方案定位为"可解释的产品手册"，强化**校园限定**的叙事；② 多屏密集展示时，每屏都对齐到同一基线（顶部状态栏底 44、TabBar 顶 49、中间内容区总宽 343），视觉网格一致；③ 状态徽章的颜色在浅/深双主题下保持完全相同的语义（报名=绿、进行=黄、预告=紫蓝），可作为后续"语义化设计 token"沉淀。

### 2.6 特别需要标记的设计细节与实现建议

> 以下 6 点是参考图中最容易"实现走样"的地方，请重点还原。

1. **课程色块的"低饱和、同明度"原则** — 8 种课程色块虽然有色相差异，但**饱和度统一在 20% 以下、明度统一在 92% 左右**，因此整体看上去协调而不是"彩虹色"。实现时把课程色定义为 `--course-math: #DCEFE2; --course-physics: #DCE6F2; …`，明度差 ≤ 5。**出处**：[14_57_50.png](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_57_50.png) 课表空闲模块。

2. **状态徽章的"底色浅 / 字色深"反白组合** — 徽章不直接用饱和色，而是浅底 + 深字（薄荷 7CD9A6 / 1A7A4A；琥珀 FFD479 / 8A5A00；蓝紫 B7C4FF / 3B47B7），保证色弱可达性。**出处**：[14_57_50.png](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_57_50.png) 校园圈活动卡片左上角。

3. **VIP 横幅"暖金渐变 + 暗夜衬底"** — 不要用纯金色实心，而用 12° 角渐变 `#C9A36A → #E8C98A`，配合深色卡片底才能产生"高级感"。**出处**：[14_57_30.png](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_57_30.png) 高级会员横幅；[14_57_39.png](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_57_39.png) 高级会员权益页。

4. **登录页"实景图占 70% 高度"** — 不要让按钮浮在图片正中而没有呼吸，要让图片占绝对高度（70%），按钮落到下方 30% 纯色区。**出处**：[14_53_26.png](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_53_26.png)、[14_57_30.png](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_57_30.png) 登录屏。

5. **最近活动 sticky banner** — 浮于 TabBar 之上、高度 64、左右内边距 12、缩略图 64×36、关闭按钮在右上，整条带 1px 顶部 8% 黑色阴影区隔。**出处**：[14_57_50.png](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_57_50.png) 首页底部。

6. **气泡 IM 的"自己无头像 / 对方有头像"** — 这是国内主流 IM（含青藤之恋）的小默契：自家消息流头像常被隐去以节省横向空间，对家消息保留 32 头像。**出处**：[14_53_26.png](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_53_26.png)、[14_59_58.png](file:///d:/6/恋爱小程序/ChatGPT%20Image%202026%E5%B9%B45%E6%9C%8817%E6%97%A5%2014_59_58.png) 聊天页。
