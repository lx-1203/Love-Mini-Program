## 严重问题

### S-001: 匹配卡片图片未加载

- **严重程度**: 严重
- **影响范围**: 匹配页面核心功能
- **问题描述**: 匹配卡片中用户头像/背景图显示 "The image is generating... Please refresh page to preview."，影响用户匹配体验
- **截图**: `usability-discover.png`
- **修复建议**:
  1. 检查图片生成服务（AI 生成头像/背景图）是否正常运行
  2. 添加图片加载失败时的兜底方案（使用默认头像/渐变背景）
  3. 增加图片加载中的骨架屏效果，替代文字提示
  4. 检查图片 URL 是否正确拼接
- **优先级**: P0（最高）
- **预计工作量**: 4-8 小时
- **技术可行性**: 高，前端兜底方案可快速实现

## 一般问题

### G-001: 管理员前端界面未开发

- **严重程度**: 一般
- **影响范围**: 系统运维管理
- **问题描述**: `apps/admin` 目录无 .vue 文件，管理功能仅通过后端 API 实现，缺乏可视化管理界面
- **修复建议**:
  1. 开发独立的管理后台前端（可基于 Vue3 + Element Plus）
  2. 实现用户管理、内容审核、认证审核、数据统计等核心管理功能
  3. 管理后台与客户端共享 Design Token 体系
- **优先级**: P1
- **预计工作量**: 40-80 小时
- **技术可行性**: 高，后端 API 已就绪

### G-002: 聊天会话页需参数才能访问

- **严重程度**: 一般
- **影响范围**: 聊天功能
- **问题描述**: 直接访问 `/pages/chat-session/chat-session` 显示空白页面，需要会话 ID 参数
- **修复建议**:
  1. 聊天会话页应从聊天列表点击进入，而非直接访问
  2. 添加参数缺失时的友好提示和返回按钮
  3. 验证从聊天列表点击会话是否正常跳转
- **优先级**: P2
- **预计工作量**: 2-4 小时
- **技术可行性**: 高

## 轻微问题

### M-001: 引导弹窗遮挡首页内容

- **严重程度**: 轻微
- **影响范围**: 首次用户体验
- **问题描述**: 首次进入首页显示 1/6 引导弹窗，遮挡主要内容
- **修复建议**:
  1. 引导弹窗设计合理，支持"跳过"操作
  2. 考虑增加"不再显示"选项
  3. 优化弹窗动画效果
- **优先级**: P3
- **预计工作量**: 2-4 小时
- **技术可行性**: 高

### M-002: 角色权限控制为简易版

- **严重程度**: 轻微
- **影响范围**: 系统安全
- **问题描述**: `AdminCertificationController` 注释说明"任何已认证用户可访问（简易版），生产环境应增加角色校验"
- **修复建议**:
  1. 添加角色权限校验（@PreAuthorize 或自定义注解）
  2. 区分普通用户和管理员角色
  3. 管理 API 仅允许 ADMIN 角色访问
- **优先级**: P1
- **预计工作量**: 4-8 小时
- **技术可行性**: 高

### M-003: 讨论圈显示"暂无热门讨论"

- **严重程度**: 轻微
- **影响范围**: 讨论圈页面
- **问题描述**: 讨论热度区域显示空状态
- **修复建议**:
  1. 空状态展示合理，可考虑添加引导发帖按钮
  2. 增加热门讨论的推荐算法
  3. 使用 mock 数据填充开发环境
- **优先级**: P3
- **预计工作量**: 2-4 小时
- **技术可行性**: 高
